## Job Description

We are seeking an experienced Senior DevOps Software Engineer to join our growing infrastructure team. You will be responsible for designing, implementing, and maintaining our cloud infrastructure, CI/CD pipelines, and deployment automation systems.

### Key Responsibilities

- Design and implement scalable cloud infrastructure on AWS and Azure
- Develop and maintain CI/CD pipelines using Jenkins, GitLab CI, and GitHub Actions
- Implement infrastructure as code using Terraform and CloudFormation
- Manage containerized applications using Docker and Kubernetes
- Implement monitoring, logging, and alerting solutions using Prometheus, ELK Stack, and Datadog
- Collaborate with development teams to optimize deployment processes
- Implement security best practices and compliance standards
- Mentor junior engineers and conduct code reviews
- Participate in on-call rotation and incident response

### Required Skills

- 5+ years of DevOps/SRE experience
- Strong proficiency in Python, Bash, and Go
- Experience with AWS (EC2, S3, RDS, Lambda, CloudFormation)
- Kubernetes and Docker containerization
- CI/CD pipeline development (Jenkins, GitLab CI, GitHub Actions)
- Infrastructure as Code (Terraform, CloudFormation, Ansible)
- Linux system administration
- Git version control
- Monitoring and logging tools (Prometheus, ELK, Datadog)
- SQL and NoSQL databases

### Preferred Skills

- Experience with microservices architecture
- Cloud cost optimization
- Security and compliance (SOC 2, HIPAA)
- Machine learning operations (MLOps)
- Kubernetes operators and Helm charts
- API gateway and service mesh experience
- Agile/Scrum methodologies
- Technical leadership and mentoring

### Nice to Have

- AWS certifications (Solutions Architect, DevOps Engineer)
- Open source contributions
- Experience with AI/ML infrastructure
- Multi-cloud experience (AWS, Azure, GCP)
- Disaster recovery and business continuity planning
